🚀 Project Overview

  This internship project focuses on building advanced AI-powered image colorization systems using Deep Learning and Computer Vision techniques. The objective was to design and implement multiple intelligent pipelines capable of transforming grayscale images into realistic color outputs across different domains.
  The project integrates modern AI frameworks such as DeOldify, YOLOv8, SegFormer, PyTorch, OpenCV, and Gradio to build end-to-end colorization systems.

🧠 Key Objectives:-
  Real-time multi-object colorization using semantic segmentation
  User-guided conditional image colorization
  Context-aware scene understanding and colorization
  Historical image restoration with time-based palettes
  Cross-domain image processing (sketches, photos, infrared images)
  Historical photograph colorization with era accuracy
  Interactive GUI development using Gradio

📌 Tasks Completed
  🔹 Task 1: Real-Time Multi-Object Colorization:
  Implemented a real-time video colorization system using YOLOv8 and SegFormer for object detection and segmentation.
  
  🔹 Task 2: Conditional Image Colorization:
  Built a user-controlled system allowing region-based color selection via GUI.
  
  🔹 Task 3: Context-Aware Scene Colorization:
  Developed a model that understands complex scenes including shadows, reflections, and spatial relationships.
  
  🔹 Task 4: Time-Based Historical Colorization:
  Applied era-specific color palettes based on historical context and user selection.
  
  🔹 Task 5: Cross-Domain Colorization:
  Handled multiple input types including sketches, grayscale photos, and infrared-style images.
  
  🔹 Task 6: Historical Photograph Colorization:
  Restored historical photographs using AI-driven realistic color generation techniques.
